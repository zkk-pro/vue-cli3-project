### 安装yarn

```javascript
npm install -g yarn
```

### 安装依赖

```
yarn install
```

### 开发环境运行

```
yarn run dev
```

### 测试环境打包

```
yarn run build:test
```

### 生产环境打包

```
yarn run build:prod
```
